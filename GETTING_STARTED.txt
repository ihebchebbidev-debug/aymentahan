🎯 5-STEP DEPLOYMENT
═══════════════════════════════════════════════════════════════════════════

STEP 1: Install Dependencies
  $ npm install
  Time: ~5 minutes

STEP 2: Build Project
  $ npm run build
  Time: ~2 minutes
  Creates: dist/ folder

STEP 3: Upload to Server
  Upload: dist/* → /code_source/
  Upload: .env → /code_source/.env
  Upload: .htaccess → /code_source/.htaccess
  Time: ~10 minutes

STEP 4: Verify Files
  ├─ index.html exists
  ├─ assets/ folder exists
  ├─ .env file in place
  ├─ backend/php/ folder exists
  └─ health.php accessible

STEP 5: Test

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: npm command not found
  Solution: Install Node.js from https://nodejs.org/

Problem: Port 5173 in use
  Solution: npm run dev -- --port 3000

Problem: Build fails
  Solution: npm install → npm run build

Problem: API returns 404
  Solution: Check .env has correct URL

Problem: Login fails
  Solution: Verify backend/php/health.php returns JSON

→ See SETUP_DEPLOYMENT_GUIDE.md for more troubleshooting
